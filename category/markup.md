### markdown 语法
- 插入一个空格 (non-breaking space)<br>
`&nbsp;    或    &#160;     或      &#xA0;
`
- 插入两个空格 (en space)<br>
`&ensp;     或    &#8194;   或      &#x2002;
`
- 插入四个空格 (em space)<br>
`&emsp;    或    &#8195;   或      &#x2003;
`
- 插入细空格 (thin space)<br>
`&thinsp;   或     &#8201;  或      &#x2009;
`
- 插入换行<br>
`<br>; 或 " SHIFT + Enter ";
`
