#### DQL
14. 查询语法

    ```
    - select [distinct] [列名，列名...] *  from 表名 where 条件;       # * 全表查询显示，[distinct]可选择去重，[列名]只查询对应列
    - select 列名 as 列别名  from 表名 as 表别名 where 条件;           #多表或复杂查询时可以使用别名
    # 注意
    - 多列去重时只有所有列都相同时才会去重，有一列不同就不会去重
    - select 列名，列名+xxx from 表名;                                # 对于列的显示进行运算，不影响原表数据
    ```
